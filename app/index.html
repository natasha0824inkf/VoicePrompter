Feature Unavailable

App-level voice recognition is blocked by Apple when VoicePrompter is installed as a home screen app on iPad.

Please open voiceprompter.app directly in Safari to use this feature.

Android Limitation

Android does not allow the camera's microphone and voice recognition to be active at the same time. You can either scroll manually or use this feature on an iPhone or desktop.

Voice Scrolling is disabled while Video Mode is turned on.
